Detailed Description

Structure of robot variables. Written by Souparni and improved upon by Omar.

This is a structure for defining the robot's variables based on the idea of a Cartesian coordinate system.

Field Documentation

float angle

Defines and records the angle of the robot. Initially set to 90.

float dist_traveled

Defines and records the total distance traveled by the robot. Initially set to zero. Reset after every movement.

char initialized

Checks if the robot is initialized. Returns 1 or 0 (True or false)

float x

Defines and records the x coordinate position of the robot. Initially set to 0.

float y

Defines and records the y coordinate position of the robot. Initially set to 0.
